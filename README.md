## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um site com múltiplas páginas, utilizando os conceitos fundamentais de front-end, como **HTML, CSS e JavaScript**, com apoio do framework **Bootstrap** para estilização e responsividade.

O objetivo foi aplicar, na prática, os conhecimentos adquiridos na disciplina de **Front-End Essencial**, incluindo estruturação de páginas, estilização, interatividade e versionamento de código com Git e GitHub.

O projeto foi desenvolvido de forma colaborativa, com organização e comunicação através do Discord.

Link: https://portalnoticiaspetropolis.netlify.app/

## 🚀 Como executar o projeto

### 📥 Clonar repositório

```bash
git clone https://github.com/anapimenta74/projeto-final-front-end.git
```

### 📂 Acessar a pasta

```bash
cd projeto-final-front-end
```

### ▶️ Executar

Abra o arquivo `index.html` diretamente no navegador.

> 💡 Dica: utilize a extensão **Live Server** no VS Code para uma melhor experiência de desenvolvimento.
