Sobre o Projeto

Este portal de notícias foi desenvolvido como projeto final da disciplina de Front-End Essencial. Utilizamos tecnologias como HTML5, CSS3 e JavaScript, integradas ao framework Bootstrap para garantir uma experiência responsiva. O trabalho foi realizado de forma colaborativa pelo Grupo 7, com foco em versionamento via Git/GitHub e comunicação via Discord.
